This is a work-in-progress game engine using OpenGL and a game called Dungeon implemented in it.
## Dependencies
### Build tools
- GCC or Clang
- Meson
- Python 3
### Libraries
- GLFW, SDL2 or OSMesa
- libconfig
- LuaJIT or Lua5.1
## Building
```
$ meson setup builddir
$ meson compile -C builddir
```
